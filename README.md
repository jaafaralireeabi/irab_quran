# irab_quran

A simple Quran study helper that provides Arabic grammar (i`rab) annotations for Quranic verses.

This project aims to:

- explain Quran text with grammatical analysis
- help learners understand Arabic syntax and morphology
- show verse-level word parsing and case endings

Usage:

1. Open the project.
2. Load a Quranic verse.
3. View the generated i`rab explanation for each word.

The README explains the purpose, features, and basic usage of the project.
